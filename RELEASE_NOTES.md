# Release Notes

## v1.0.1

发布日期：2026-03-19

### 本次更新

- 调整 AI 启用策略：服务启动后默认具备 AI 能力，用户点击成员详情中的“生成 AI 分析”按钮后再调用 DeepSeek。
- 优化列表体验：成员档案列表、健康记录列表、预警列表统一改为每页 10 条分页展示。
- 精简成员详情页 AI 区域文案，减少与业务无关的说明性提示。
- 新增本地配置文件方案：支持通过 `config/application.properties` 在本机保存 DeepSeek 配置，且默认不会提交到 GitHub。
- 重构 README：补充项目目的与意义、功能分层、技术路线、核心流程、目录结构、运行方式、AI 配置说明等内容。

### 适用场景

- Spring Boot 课程设计
- 毕业设计原型展示
- 家庭健康管理类系统演示
- AI + 医疗健康方向功能扩展示例

### 相关备注

- 公开仓库不包含真实 DeepSeek API Key。
- 建议本地通过 `config/application.properties` 或环境变量注入 API Key。
- AI 分析仅作为辅助建议，不替代医生诊断。
- H2 为本地文件数据库，`data/` 目录不建议提交到 GitHub。

## v1.0.0

发布日期：2026-03-18

### 版本功能

- 新增家庭成员健康档案管理，支持关系、出生日期、血型、职业、吸烟饮酒情况、慢病史、用药史、手术史、过敏史、运动习惯和健康目标等信息维护。
- 新增更完整的健康记录维度，覆盖体重、腰围、血压、空腹血糖、餐后血糖、体温、血氧、胆固醇、睡眠、运动、步数、饮水、情绪、压力、症状与用药记录。
- 新增疾病风险评估与预警机制，可根据异常指标自动计算风险等级并生成预警建议。
- 新增成员详情页，支持点击进入具体成员档案，查看基础信息、最近记录、最近预警与健康摘要。
- 新增趋势追踪能力，展示体重、收缩压、空腹血糖、睡眠时长、运动时长、风险分数等指标变化趋势。
- 新增个性化健康建议，根据最新风险状态和近期趋势生成可读性更强的生活方式建议。
- 新增 DeepSeek AI 咨询能力，用户主动点击后才会触发 AI 分析，避免自动消耗 token。
- 新增一键启动脚本 `start-app.bat`，支持启动后自动打开浏览器首页。

### 适用场景

- Spring Boot 课程设计
- 毕业设计原型展示
- 个人或家庭健康信息管理演示
- AI + 医疗健康方向功能扩展示例

### 相关备注

- 为避免敏感信息泄露，公开仓库版本不包含真实 DeepSeek API Key。
- AI 功能需通过环境变量手动启用：
  - `DEEPSEEK_ENABLED=true`
  - `DEEPSEEK_API_KEY=你的Key`
- AI 咨询仅作为健康分析辅助，不替代医生诊断。
- H2 使用本地文件数据库，运行过程中生成的数据默认不建议提交到 GitHub。
- 当前系统为单机演示版，尚未接入登录鉴权、角色权限和正式生产级安全策略。
